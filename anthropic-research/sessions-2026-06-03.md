### Summary
Anthropic launched the Claude Compliance API on May 21, giving enterprise IT and security teams programmatic access to two data streams: full conversation content (chats, uploaded files, projects) and activity events (logins, admin actions, configuration changes). The API now integrates with 28 providers spanning DLP, SASE, SIEM, identity management, eDiscovery, and AI security posture management — including Cloudflare, CrowdStrike, Microsoft Purview, Okta, Palo Alto Networks, Proofpoint, Snyk, Wiz, and Zscaler. Days later, a separate security-guidance plugin for Claude Code was released, intercepting file edits in real time against 25 known vulnerability patterns before any code reaches a commit, delivering a 30–40% reduction in security-related PR review comments. Both capabilities are available immediately: the Compliance API for Enterprise plans, the security-guidance plugin free for all users.
