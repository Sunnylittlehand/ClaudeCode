### 摘要
Code with Claude 2026旧金山站于5月6日举行，Anthropic发布了Claude托管智能体三项重大新功能：多智能体编排（Orchestration）支持将复杂任务分解并行处理；成果导向评估（Outcomes）允许用户定义成功标准，智能体据此自主迭代；"梦境"（Dreaming）则让智能体在会话间持续学习、沉淀记忆。主创Boris Cherny宣称"默认模式已从'我去提示Claude'转变为'让Claude自我提示'"，并预示无限上下文窗口即将到来。大会未发布新模型，而是聚焦于如何将现有模型推向生产级自主工作流——这是Anthropic向智能体编排平台转型的标志性宣言。

### Summary
Anthropic's Code with Claude 2026 conference in San Francisco (May 6) served as the company's definitive declaration of its agentic platform strategy. Three major capabilities shipped to public beta under Claude Managed Agents: **Multiagent Orchestration** (a lead agent breaks work into pieces and delegates to specialist subagents running in parallel on a shared filesystem); **Outcomes** (users write success rubrics and a separate grader evaluates agent output, lifting task success by up to 10 points on hard problems); and **Dreaming** (a scheduled process that reviews agent sessions, extracts patterns, and curates persistent memory so agents improve over time). Boris Cherny framed the paradigm shift bluntly: "The default isn't 'I'm going to prompt Claude' — the default is now 'I'm going to have Claude prompt itself.'" No new model was announced; the message was that production-grade agentic automation of existing models is the frontier. Dianne Penn hinted at "context windows that feel infinite" as a near-term development.

### Main Takeaway
Anthropic has crossed the threshold from AI assistant to autonomous orchestrator — Claude Managed Agents with Dreaming and Outcomes means the model is now designed to manage itself across multi-session, multi-agent enterprise workflows.
