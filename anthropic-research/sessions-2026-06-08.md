### Significance Rating
⭐⭐⭐⭐⭐ — Dual release: a meaningfully improved flagship model and a genuinely new architectural capability (Dynamic Workflows) that changes the ceiling of what a single Claude Code session can accomplish.

### 摘要
Anthropic于5月28日发布Claude Opus 4.8，并同步推出"动态工作流"功能。新模型在代码缺陷自我识别方面表现显著提升——错误漏报率较4.7版下降约75%，在线网页代理基准得分达84%，超越GPT-5.5。"动态工作流"允许Claude Code在单次会话中协调最多1000个并行子代理，整个工程流程通过JavaScript脚本而非上下文窗口来管理，上限为16个并发代理。Bun创始人以此重写了约100万行代码，仅耗时6天。

### Summary

Simultaneously, Anthropic shipped Dynamic Workflows into research preview for Claude Code: a feature that lets a single session spawn up to 1,000 subagents (16 concurrent), with orchestration logic living in a Claude-authored JavaScript script rather than the context window — enabling tasks at a scale previously impossible in a single run. The feature requires Claude Code v2.1.154+ and is available on Max, Team, and Enterprise plans.

### Main Takeaway
