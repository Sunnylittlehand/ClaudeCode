### Summary
Anthropic's annual Code with Claude 2026 conference (San Francisco, May 6) shipped three production-ready upgrades to Claude Managed Agents: **Dreaming** — a background memory-consolidation process that reads past sessions and writes notes on recurring mistakes, convergent workflows, and team-wide preferences, so agents improve between jobs without developer intervention; **Outcomes** — a self-grading loop where a separate evaluator agent scores output against a developer-written rubric and returns a structured critique, closing the gap where agents complete tasks incorrectly without realising it; and **Multiagent Orchestration** — a native coordinator that decomposes complex tasks and fans them to specialist subagents running in parallel. On the infrastructure side, Anthropic announced a massive compute partnership with SpaceX Project Colossus (220,000 Nvidia GPUs, 300 MW), removed peak-hour throttling for Pro and Max accounts, and doubled five-hour usage limits. For enterprise customers, a new security tool (powered by Opus 4.7) scans entire code repositories for vulnerabilities and proposes fixes. The desktop GUI was also redesigned with split views, pinnable chapter markers, and inline diff comments.
