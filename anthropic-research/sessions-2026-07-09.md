## [China's MIIT Issues Formal "Backdoor" Security Alert Over Anthropic's Claude Code — And Anthropic Confirms Hidden Tracking Code Was Real](https://www.cnbc.com/2026/07/08/china-anthropic-ai-claude-code-backdoor-security-threat.html)

### Speaker / Author
China's Ministry of Industry and Information Technology (MIIT) via its cybersecurity threat platform; Anthropic's engineering team via X posts and official response. This is a geopolitical and software-integrity story with direct enterprise implications.

### Significance Rating
⭐⭐⭐⭐ — Anthropic confirmed it embedded undisclosed tracking code in Claude Code to detect Chinese users, a developer found it, China formally labelled it a "backdoor," and Anthropic removed the code — a software supply-chain integrity incident with FCA/PRA third-party risk implications.

### 摘要
中国工业和信息化部对 Anthropic 的 Claude Code（版本 2.1.91 至 2.1.196）发出正式安全警告，将其列为含"后门漏洞"的工具，可将用户位置和身份信息传送至远程服务器。Anthropic 确认，代码中确实存在一段未公开的跟踪机制——当检测到中国时区（Asia/Shanghai 或 Asia/Urumqi）或中国代理域名时，会通过"提示词隐写术"（prompt steganography）修改系统提示中的日期格式，向 Anthropic 后端发送隐蔽标记，目的是识别规避访问限制的非法用户及模型蒸馏行为。该代码已于 7 月 1 日版本中移除。

### Summary
China's MIIT formally warned that Claude Code versions 2.1.91–2.1.196 (April 2 – June 29 releases) contain a "security backdoor vulnerability that poses a serious threat," capable of sending user location and identity data to remote servers. Anthropic confirmed the code existed: an undisclosed experimental anti-abuse mechanism that used "prompt steganography" — encoding a Chinese timezone or proxy detection signal inside the "Today's date is…" system prompt string — to flag suspected distillation and reseller abuse. A developer known as "Thereallo" discovered the mechanism by inspecting minified JavaScript; Anthropic removed it in the July 1 Claude Code release and described it as "an experiment we launched in March." Affected versions span approximately 2.1.91 to 2.1.196; the current version 2.1.204+ is clean. Anthropic reiterated that Claude access is not permitted in China.

### Main Takeaway
Anthropic quietly embedded undisclosed tracking code in Claude Code for four months; a public discovery forced a disclosure and removal, creating a software supply-chain integrity story that every enterprise security team will need to assess.
