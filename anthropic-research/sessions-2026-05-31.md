### Significance Rating
⭐⭐⭐⭐ — This is the most directly enterprise-security-relevant product release of the month: it resolves the most commonly cited compliance objection to Claude Managed Agents by letting enterprises run tool execution entirely inside their own infrastructure perimeter.

### 摘要
Anthropic于5月19日推出自托管沙箱（公测版）和MCP隧道（研究预览版）。自托管沙箱允许企业将智能体的工具执行层部署在自有基础设施上，数据不离开企业边界，现有审计日志、DLP控制和身份系统自动生效。MCP隧道则让Claude智能体无需公开暴露内部服务，即可通过单一出站加密连接访问私有API和数据库。两项功能直接解决了受监管行业在采用AI智能体时面临的核心数据驻留和访问控制问题。

### Summary
Anthropic announced self-hosted sandboxes (public beta) and MCP tunnels (research preview) for Claude Managed Agents on 19 May 2026. Self-hosted sandboxes allow enterprises to move the tool-execution layer of AI agents onto their own infrastructure or managed providers (Cloudflare, Daytona, Modal, Vercel), while the agent loop and orchestration remain on Anthropic's servers. This means files, repositories, and runtime packages never leave the enterprise perimeter, and existing network policies, audit logging, DLP controls, and identity systems apply automatically. MCP tunnels complement this by allowing agents to connect to private Model Context Protocol servers — internal databases, private APIs, ticketing systems — via a single outbound end-to-end encrypted connection with no inbound firewall rules required. Both features were announced at the Code with Claude London event.

### Main Takeaway
Self-hosted sandboxes resolve the single biggest compliance blocker for enterprise Claude Code adoption: code and data now stay inside the customer's own infrastructure boundary.

**Anthropic Editorial** — Released via the Claude Developers account and the Anthropic news blog. Available to all Claude Code users across all plan tiers.

### Significance Rating

### 摘要
Anthropic于5月27日发布Claude Code安全指导插件，通过/plugins命令即可安装。该插件实时监控代码编辑、差异比较和提交内容，自动检测SQL注入、命令注入、XSS、硬编码API密钥等约25类高危漏洞模式，并在同一编码会话中提供内联修复建议，无需切换至独立安全扫描工具。内部数据显示，插件上线后拉取请求中安全相关审查意见减少30-40%。该插件对所有计划层级用户免费开放。

### Summary
The Claude Code security-guidance plugin, released on 27 May 2026, is a free tool available to all plan tiers via the /plugins marketplace. Once installed, it monitors code edits, diffs, and commits in real time using regex-based pattern matching against approximately 25 high-risk vulnerability classes, including SQL injection, command injection, XSS, hardcoded secrets, insecure deserialization, and improper input validation. When a risky pattern is detected, Claude prompts an inline fix within the same coding session — eliminating the context-switch to a separate security scanner. Internal data reportedly shows a 30–40% reduction in security-related pull request comments since introduction.

### Main Takeaway
