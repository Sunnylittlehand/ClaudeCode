Anthropic Editorial. Claude Cowork is Anthropic's desktop-first AI agent for knowledge work, launched in January 2026. ### Significance Rating
⭐⭐⭐⭐☆ — Cowork's cross-device expansion reframes it from a developer-adjacent tool into a general enterprise AI platform, and Anthropic's usage data showing that software development is only 8.7% of sessions is a landmark disclosure that will reshape how boards and regulators think about AI-driven workforce change.

### 摘要
Anthropic 将 Claude Cowork 扩展至网页版和移动端，首批向 Max 用户开放。配套发布的使用数据揭示：在 120 万匿名会话中，业务流程处理占比 33.4%、内容创作占 16.4%，而软件开发仅占 8.7%——AI 助手的主战场已不在代码编辑器，而在整个办公室。新功能"Dispatch"可在笔记本合盖后持续运行任务，并在手机上同步状态。

### Summary

Sessions are stored remotely in beta, so work persists across devices with no local state required.

### Main Takeaway

**Type**: Product Update / API Model Release
**Date**: 2026-07-06
**Links**: [OpenAI announcement](https://openai.com/index/introducing-gpt-realtime/) · [OpenAI developer community](https://community.openai.com/t/new-realtime-models-on-the-api-gpt-realtime-2-1-and-gpt-realtime-2-1-mini/1385896) · [TechTimes](https://www.techtimes.com/articles/319860/20260707/openai-realtime-api-cuts-voice-agent-latency-25-adds-reasoning-mini-model.htm)

### Speaker / Author
OpenAI Editorial. The Realtime API is OpenAI's speech-to-speech interface for building low-latency voice agents that respond to audio over a live WebSocket connection, and gpt-realtime-2.1 is the latest generation of production-grade voice models for that platform.

### Significance Rating
⭐⭐⭐☆☆ — Meaningful capability lift for builders of voice-based AI agents, but limited direct relevance to Claude Code or text-based enterprise AI; the 25% p95 latency reduction and the new mini reasoning model that talks while executing tool calls are real engineering improvements.

### 摘要
OpenAI 发布 gpt-realtime-2.1 和 gpt-realtime-2.1-mini，全线语音模型 P95 延迟降低 25%，mini 版可在调用工具时持续输出语音，彻底消除通话静默。定价：mini 音频输入 $10/百万 token（全量版 $32），输出 $20（全量 $64）。

### Summary
OpenAI released gpt-realtime-2.1 and gpt-realtime-2.1-mini on July 6, reducing p95 latency by at least 25% across all Realtime voice models via improved caching. The full model improves alphanumeric recognition (order numbers, phone numbers, confirmation codes), silence and noise handling, and interruption detection. The mini model is a distilled reasoning model that continues speaking during tool-call execution — eliminating the dead silence that makes callers assume a line has dropped, a key friction point in production voice agents. Pricing: gpt-realtime-2.1-mini at $10/$20 audio in/out per million tokens versus $32/$64 for the full model.

### Relevance to Sunny
⭐⭐☆☆☆ Low relevance — general interest. Voice agent infrastructure is not in scope for Sunny's current Claude Code rollout. However, as regulated financial services increasingly explore voice-based AI (trading desk assistants, client-service agents), this release signals the direction of travel for OpenAI's platform and helps Sunny understand what OpenAI is building toward.
